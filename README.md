# Qrify - Advanced QR Code Generator  

🌐 **Project Website:** [https://www.qr.frameworkteam.com/](https://www.qr.frameworkteam.com/)  

This app is built with **Lovable AI** and **Antigravity AI**, combining modern AI-assisted development with a powerful full-stack architecture to deliver a seamless QR code generation experience.

Qrify is a robust, full-stack web application designed for generating, customizing, and managing dynamic QR codes. It offers advanced styling options such as custom colors, unique eye shapes, and embedded logos, along with user management features for saving and tracking QR code history.

---

## 🚀 Tech Stack  

This project is built using modern web development tools and best practices:

### 🧩 Core Frontend  
- **React 18** – Core UI library  
- **Vite** – Fast build tool & dev server  
- **TypeScript** – Type-safe JavaScript  
- **React Router DOM** – SPA routing  

### 🎨 Styling & UI  
- **Tailwind CSS** – Utility-first styling  
- **Radix UI + shadcn/ui** – Accessible UI primitives  
- **Lucide React** – Icon library  

### 🔄 State Management & Forms  
- **TanStack React Query** – Data fetching & caching  
- **React Hook Form** – Form management  
- **Zod** – Schema validation  

### 🛠 Backend Infrastructure  
- **Supabase**
  - PostgreSQL Database  
  - Authentication system  
  - Row Level Security (RLS)  
- **Supabase JS Client** – API interaction  

---

## 🎯 QR Code Features  

Qrify provides powerful QR code customization using:

### ✨ Libraries  
- **qr-code-styling**
  - Custom colors (foreground/background)  
  - Unique dot & eye styles  
  - Logo embedding  

- **qrcode.react**
  - Lightweight QR generation  
  - Responsive rendering  

---

## 📦 Export & Rendering  

- **html-to-image** – Export as PNG/JPEG  
- **jsPDF** – Export as PDF (print-ready)
